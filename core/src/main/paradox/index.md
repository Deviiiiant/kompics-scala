# Kompics Scala

The Scala DSL for the [Kompics](https://kompics.sics.se) component framework.

## Setup

Add the following dependency to your `build.sbt` file:

@@@vars
~~~ scala
libraryDependencies += "se.sics.kompics" %% "kompics-scala" % "$project.version$"
~~~
@@@
